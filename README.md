# Interview AI Prep

Interview AI Prep is an AI-powered interview preparation platform designed to help users practice technical and behavioral interviews through realistic mock interview sessions. The platform provides AI-generated questions, instant feedback, performance analysis, and personalized recommendations to improve interview skills and confidence.

## 🚀 Features

* AI-powered mock interviews
* Technical and behavioral question generation
* Real-time interview simulation
* Instant feedback and performance analysis
* Personalized improvement recommendations
* Interview history and progress tracking
* User authentication and profile management
* Responsive design for all devices

## 🛠️ Tech Stack

### Frontend

* React.js
* TypeScript
* Tailwind CSS
* Next.js

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* OpenAI API

## ⚙️ Installation

```bash
git clone https://github.com/kartiktalwar2006-cmd/interview-ai-prep.git
cd interview-ai-prep
```

Install dependencies:

```bash
cd client && npm install
cd ../server && npm install
```

Run the project:

```bash
# Frontend
npm run dev

# Backend
npm start
```
